## 规范

构建和同步配置文件的命名规范为

- `Dockerfile-${tag}`
- `${tag}.yml`

工作流文件的规范为

- 工作流 ID 格式为相对路径,以 `-` 分割

- 名字为 `构建 $ID 镜像`
- job 为 `build-$ID`
- 文件名为 `$ID`
